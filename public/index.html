About the Project

This project demonstrates the integration of Augmented Reality using A-Frame and AR.js. By scanning a marker, you can view a YouTube video embedded within the AR scene.

How to Use

  1. Open this page on a device with a webcam or a mobile device.
  2. Allow the webpage to access your camera.
  3. Click the button below to start the AR demo.
